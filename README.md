### Parser
[http-parser](https://github.com/nodejs/http-parser)
这个解析库在2019年已经停止，原因是作者觉得这个库已经难以维护，转为使用
[llhttp](https://github.com/nodejs/llhttp)
使用 TypeScript 编写，生成 C 代码，benchmark 显示生成的 C 代码运行比 http-parser 更快
